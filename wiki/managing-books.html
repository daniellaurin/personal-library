MANAGING BOOKS

BUILD YOUR COLLECTION

SEARCHING FOR BOOKS (RECOMMENDED)

The fastest way to add books with full details and cover images:

  1. Click SEARCH in the navigation bar.
  2. Type a title, author name, or ISBN in the search box.
  3. Press Enter or click SEARCH.
  4. Results from the Google Books API appear as cards.
  5. Click + ADD TO LIBRARY on any result to add it instantly.
  6. You will be taken to the book's detail page where you can write a review.
If the book is already in the library, clicking "Add to Library" will take you straight to its existing page instead of creating a duplicate.

MANUAL BOOK ENTRY

To add a book that is hard to find in the API, use manual entry:

  1. Go to LIBRARY and click + MANUAL ENTRY (requires login).
  2. Fill in the title, author, publisher, and year.
  3. Click ADD TO LIBRARY.

Manual entries will not have a cover image unless one is added via the API search later.

BROWSING THE LIBRARY

Click LIBRARY in the nav to see all books in the collection, sorted by most recently added. Each card shows:

  • Book cover image (from Google Books API)
  • Title and author
  • Average community rating (if reviews exist)

Click any card to open the book's detail page with full metadata and all reviews.

UPDATING OR DELETING BOOKS (ADMIN ONLY)

Admin users can manage the catalogue from the Admin Books page:

  • Delete - removes the book and all associated reviews permanently.
  • Edit author - use the Edit Book page to update the author name.
Deleting a book is irreversible. All reviews for that book will also be deleted.

UPDATING SITE IMAGES AND MEDIA

To update the images and audio used by the site:

  • Cover images - come from the Google Books API automatically. No manual uploads needed.
  • Background / UI images - replace files in the /images/ folder. Keep the same filenames.
